# Backend for online-store

## Microservices list

- shop
- auth
- payments

---

- api-gateway

## Install

1. Install dependencies in each microservice

```bash
./install.sh
```

2. Add values to .env files in each microservice

## Development

To run all microservices in development mode:

```bash
npm run develop
```

To run api-gateway

```bash
npm run proxy
```
